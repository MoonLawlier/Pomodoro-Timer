# 🌙 Pomodoro Focus Timer

A minimal, aesthetic Pomodoro timer designed to help you focus deeply while enjoying a cozy vibe ✨

# 🧠 About the Project

This is a simple yet visually pleasing Pomodoro Timer Web App built to improve focus and productivity.

I built this when I was bored… and it turned into something actually useful.

# ✨ Features
* ⏱️ 25-minute Pomodoro session
* ☕ Short & long break modes
* ▶️ Start / Reset controls
* 🌌 Aesthetic UI with cozy background
* 🧘 Clean, distraction-free design

3 🛠️ Tech Stack
* HTML
* CSS
* JavaScript
